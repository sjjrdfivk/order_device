生命周期
- initState: 初始化状态
- didChangeDependencies: state依赖关系变更
- build: 构建视图
- didUpdateWidget: 状态变更，重新渲染视图
- deactivate: ui 组件被暂时移除（如切换页面）
- dispose: ui 被永久销毁
